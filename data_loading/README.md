# PyTorch DATA LOADING AND TRANSFORMATION TUTORIALS
A repository showcasing examples of using PyTorch torch.utils.data.Dataset class
# Prerequisites
- PyTorch 0.4.1
- TorchVision 0.2.1
- Pillow 4.1.1
- Matplotlib 2.0.2
